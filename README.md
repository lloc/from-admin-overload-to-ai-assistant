# From Admin Overload to AI Assistant

Scale WordPress with intelligence.

## Requirements

- PHP >= 8.3
- WordPress >= 6.9

## Installation

```bash
composer install
```

## Development

Install wp-env globally:

```bash
npm -g install @wordpress/env
```

Start the local environment:

```bash
wp-env start
```
